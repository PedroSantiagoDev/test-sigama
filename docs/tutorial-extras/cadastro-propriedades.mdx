# Cadastro de Propriedades

Aprenda como cadastrar e gerenciar suas propriedades rurais no SIGAMA.

## Visualizar Propriedades

No menu lateral, acesse **Cadastros → Propriedades**. Você verá a lista de propriedades vinculadas ao seu CPF/CNPJ.

## Informações da Propriedade

Cada propriedade contém:

| Informação | Descrição |
|------------|-----------|
| Inscrição Estadual | Número único da propriedade na SEFAZ-MA |
| Endereço | Localização completa da propriedade |
| Área | Tamanho da propriedade em hectares |
| Situação Cadastral | Status do cadastro (ativo, pendente, etc.) |

## Atualizar Dados

Alguns dados podem ser atualizados diretamente pelo sistema. Para alterações cadastrais maiores, procure uma Unidade Local da AGED-MA.

## Vincular Propriedade

Caso uma propriedade não apareça na sua lista, verifique:

1. Se a Inscrição Estadual está vinculada ao seu CPF/CNPJ na SEFAZ-MA
2. Se o cadastro da propriedade está ativo

:::tip[Dica]
Mantenha os dados da propriedade sempre atualizados para evitar problemas na emissão de documentos.
:::
