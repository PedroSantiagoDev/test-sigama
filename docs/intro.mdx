# Bem-vindo aos Tutoriais SIGAMA

Aprenda a utilizar o **SIGAMA - Sistema de Gestão Agropecuária do Maranhão**.

## O que é o SIGAMA

O SIGAMA é o sistema oficial da [AGED-MA](https://aged.ma.gov.br) (Agência Estadual de Defesa Agropecuária do Maranhão) para gestão de propriedades rurais, emissão de guias, declarações e controle agropecuário.

## Primeiros Passos

### Criar conta no SIGAMA

1. Acesse [sigama.aged.ma.gov.br](https://sigama.aged.ma.gov.br)
2. Clique em **Primeiro Acesso**
3. Selecione o tipo de usuário (Produtor Rural, Empresa, etc.)
4. Informe seu CPF/CNPJ e Inscrição Estadual
5. Siga as instruções na tela para concluir o cadastro

### Fazer login

1. Acesse [sigama.aged.ma.gov.br](https://sigama.aged.ma.gov.br)
2. Informe seu CPF/CNPJ no campo **Usuário**
3. Informe sua senha
4. Clique em **Entrar**

## Recuperar Senha

Caso tenha esquecido sua senha:

1. Na página de login, clique em **Recuperar Acesso**
2. Siga as instruções para redefinir sua senha
