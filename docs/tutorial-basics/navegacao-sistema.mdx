# Navegação no Sistema

Conheça a interface principal do SIGAMA após o login.

## Tela Inicial

Após fazer login, você verá a **página inicial** do SIGAMA com:

- Menu lateral de navegação
- Painel de notificações e avisos
- Atalhos para as funcionalidades mais usadas

## Menu Principal

O menu lateral organiza as funcionalidades do sistema:

| Seção | Funcionalidades |
|-------|----------------|
| **Cadastros** | Propriedades, produtores, rebanhos |
| **Sanidade** | Vacinações, exames, declarações |
| **Trânsito** | Emissão de GTA, guias de trânsito |
| **Relatórios** | Consultas, extratos, declarações |
| **Configurações** | Dados cadastrais, senha, e-mail |

## Barra Superior

No topo da tela você encontra:

- Nome do usuário logado
- Acesso rápido às configurações da conta
- Botão para sair do sistema

:::tip[Navegação Rápida]
Use os atalhos na página inicial para acessar rapidamente as funções mais utilizadas.
:::
